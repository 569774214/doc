# 使用

[多种效果](https://animejs.com/documentation/#cssSelector)
[中文网](http://www.animejs.cc/documentation/)

```
anime({
  targets: 'div',
  translateX: 250,
  rotate: '1turn',
  backgroundColor: '#FFF',
  duration: 800
});
```
