
# Math 对象

用于执行数学任务

Math 对象不像Date 和 String 那样的对象的类，因此没有构造函数Math()

## 语法

var x = Math.PI;            // 返回PI
var y = Math.sqrt(16);      // 返回16的平方根

## 属性

PI                  圆周率


## 方法

ceil(x)             x向上取整
floor(x)            x向下取整
pow(x,y)            返回x的y次幂
max(x,y,z,...n)     返回最大的值
min(x,y,z,...n)     返回最小的值
random()            返回0-1之间的随机数
round(x)            四舍五入
sqrt(x)             返回x的平方根
abs(x)              返回x的绝对值

sin(x)              x的正弦
tan(x)              x的正切
cos(x)              x的余弦
acos(x)             x的反余弦值
asin(x)
atan(x)
