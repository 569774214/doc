
# vue-loader

vue-loader 是 webpack 的加载器

允许您以称为单文件组件（SFC）的格式创作Vue组件：

```
	<template>
	  <div class="example">{{ msg }}</div>
	</template>

	<script>
	export default {
	  data () {
	    return {
	      msg: 'Hello world!'
	    }
	  }
	}
	</script>

	<style>
	.example {
	  color: red;
	}
	</style>
```
