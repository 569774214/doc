### 1. each

  each() 方法规定为每个匹配元素规定运行的函数。

提示：返回 false 可用于及早停止循环。

```

	$(selector).each(function(index,element))

```

```
	  $("li").each(function(){
	    alert($(this).text())
	  });
```
